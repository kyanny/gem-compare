# Gem::Compare

Compare gem source code

## Installation

Add this line to your application's Gemfile:

    gem 'gem-compare'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install gem-compare

## Usage

    $ bundle exec gem compare bundler 2.0.0...2.0.1
